# MamunFood — Static Site

A minimal static site to showcase menu and info.
Plain HTML, CSS, optional JS.

## Features

- Responsive layout
- Fast, no build step
- Easy to deploy (GitHub Pages / Netlify)

## Quick start

1. Clone or copy files into repo root.
2. Open `index.html` in browser.
3. Live link: `https://www.mamunfood.com/` and visit `https://mamun-food-mb7tjgkxy-joy094s-projects.vercel.app/`.

## Structure

- index.html
- /assets (images, fonts)
- /css (styles)
- /js (scripts)

## Deploy

- Push to GitHub and enable Pages, or
- Drag & drop to Netlify/Vercel
